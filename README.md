# EasyJapanese — Roadmap Dự Án

Website tự học tiếng Nhật: ghi chú từ vựng, luyện viết chữ, luyện nghe, ngữ pháp và flashcard.

## Tech Stack
- **Framework**: Next.js 16 (App Router) + TypeScript
- **Styling**: Tailwind CSS v4 + Shadcn UI
- **Notifications**: Sonner
- **State**: Zustand + localStorage
